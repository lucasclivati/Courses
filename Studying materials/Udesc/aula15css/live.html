HTML

A linguagem HTML é a linguagem de marcação padrão para as páginas da Web.

Com HTML é possível criar o seu próprio site.

Tag Descrição Exemplo
<p> A tag P define um novo parágrafo <p>Isto é um parágrafo.</p>
<a> A tag A define um novo link <a href="url">Texto do link.</a>
<h1> A tag h1 define um novo título com importância 1 <h1>Texto do link.</h1>

CSS

CSS é a linguagem usada para estilizar um documento HTML.

O CSS descreve como os elementos HTML deverão ser exibidos.

Propriedade Descrição Exemplo
background-color A propriedade background-color especifica a cor de fundo de um elemento p { background-color: blue}
background-image A propriedade background-image especifica uma imagem para utilizar como plano de fundo de um elemento body { background-image: url("caminho_imagem.jpg") }
color A propriedade color especifica a cor da fonte dos elementos. h1 { color: red }
text-align A propriedade text-align especifica o alinhamento do texto. p { text-align: center }
font-family A propriedade font-family especifica qual a fonte será utilizada. body { font-family: verdana; }
